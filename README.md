Instructions Portfolio – WRT 307 Service Learning  
Last Edit: 2026-03-24 
============================================================

This project was created by Soleil Cohn as part of a group Instructions Project for WRT 307: Writing for the Professions at Syracuse University.

This project involved collaborating with a team to design and develop a set of digital, illustrated instructions for assembling a custom Lego-based object. The purpose of the project was to create clear, user-friendly instructions that effectively combine written steps with visual elements.

This repository contains the portfolio website developed to present the team’s instructional content and project components.

Live Site:  
https://soleilcohn.github.io/Instructions-Portfolio/

----------------------
PROJECT PURPOSE:
----------------------

The objective of this assignment was to create a clear, concise, and visually supported instruction set for a Lego-based design. The project required teams to develop instructions that are easy to follow, visually effective, and usable by a real audience.

The project emphasizes:
- Writing clear, step-by-step instructions  
- Using visuals as communication tools, not decoration  
- Designing user-friendly digital interfaces  
- Conducting usability testing and revising based on feedback  
- Collaborating effectively in a team environment  

----------------------
WORK COMPLETED:
----------------------

As part of this collaboration, the project team:

- Designed a custom Lego model consisting of multiple components  
- Created step-by-step instructional content for assembly  
- Developed visual aids including labeled images and diagrams  
- Conducted usability testing to evaluate clarity and effectiveness  
- Revised instructions based on testing feedback and user performance  

As an individual contribution, I:

- Designed and developed the portfolio website to present the project  
- Structured content for clear navigation and usability  
- Integrated images, instructions, and team materials into a cohesive layout  
- Ensured the website met assignment requirements for accessibility and design
